📄**Description**

Main objective is to practice and learn is about practice and learn Abstract Factory Design Pattern


### English
Create a small international address manager and telephone number manager.

The application should allow you to add to the agenda, addresses and international telephone numbers. Taking into account the different formats of the different countries, build the agenda, addresses and telephones by implementing an abstract factory pattern.

### Català
Crea un petit gestor de direccions i números de telèfon internacionals.

L'aplicació ha de permetre afegir a l'agenda, adreces i números de telèfon internacionals. Tenint en compte els diferents formats dels diferents països, construeix l'agenda, les adreces i els telèfons implementant un patró Abstract Factory.
